❓ 율정7단지 하자·자재 관리 앱 설명서
📱 앱 개요
코티스 엑셀 자료를 업로드하면 세대별 하자 이력·진행상태·변화를 한눈에 파악할 수 있어요. 공가앱과 Firebase로 실시간 연동됩니다.
🚀 처음 시작하는 순서
1
🔥 Firebase 연결 클릭 → 공가앱 데이터 자동 연동 (공가/불가 세대 정보)
2
📡 코티스 업로드 클릭 → 엑셀 파일 선택 (여러 개 한번에 가능) → 자동 저장
3
상단 칩 또는 왼쪽 필터 탭으로 확인할 세대 선택
4
세대 클릭 → 상세 확인 후 필요시 메모 입력 또는 ✅ 확인완료
🎨 경과일 색상 의미
🔴 7일+ 접수 후 1주일 경과 → 업체에 독촉 전화 필요 🟠 30일+ 한 달 이상 미처리 → 진행상황 확인 필요 🔵 90일+ 3개월 이상 방치 → LH 담당자에게 확인 필요 ⚫ 180일+ 반년 이상 방치 → 강하게 클레임 필요
📋 탭별 기능 설명
📋 미처리 목록
현재 처리 안 된 하자가 있는 세대 전체 목록
• 세대 클릭 → 접수번호·하자내용·처리내용·민원인·연락처·업체명·날짜흐름 전체 확인
• 날짜 흐름: 접수→지시→예정→완료보고→완료 단계별 표시, 예정일 지나면 빨간색
• 미처리만 펼침, 완료 건은 접혀 있음 (클릭하면 펼쳐짐)
• 📞 전화번호 클릭하면 바로 전화 연결
• 🟡 변화 감지: 이전 업로드 대비 상태가 바뀐 세대 표시
• ✅ 확인완료: 누르면 🟡 목록에서 제거, Firebase에 저장
🔄 변화 이력
코티스 업로드할 때마다 상태가 바뀐 세대 기록
• 날짜별로 묶어서 표시 (예: 이관접수 → 보수처리중)
• 코티스를 2회 이상 업로드해야 변화가 감지됨
• 클릭하면 해당 세대 상세로 바로 이동
📊 분석
전체 하자 현황을 다양한 기준으로 분석
• 하자구분별: 유지보수·관리소처리 등 구분별 현황
• 보수구분별: 일상보수·자재사용 등 구분별 현황
• 동별 미처리: 어느 동에 미처리가 많은지
• 미처리 Top10: 하자가 가장 많이 쌓인 세대 순위
🔧 자재 관리
91개 품목 재고 관리
• 초기 재고: 엑셀 자료 기준으로 자동 입력됨
• 자동 차감: 코티스 업로드 시 보수구분=자재사용 건에서 품명 매칭해서 자동 차감
• + 입고: 새로 구입한 자재 직접 수량 입력
• - 출고: 수동으로 사용 수량 차감
• ⚠️ 미매칭: 자동 매칭 안 된 항목 → 직접 확인 후 수동 출고 처리
• 재고 5개 이하 → 노란색, 0개 → 빨간색 경고
💡 코티스 업로드 팁
• 파일명에 날짜 포함하면 자동 인식: 20260320_양주옥정7.xls
• 날짜 없는 파일은 상단 📅 날짜 입력칸에서 직접 선택
• 여러 파일 한번에 업로드 가능 (파일명 순서대로 처리)
• 2~4일마다 업로드하면 변화 감지가 가장 효과적
• 처음 올릴 때 자재 재고 자동 차감은 오늘 이후 자료부터 적용됨
🔗 공가앱 연동
• 🔥 Firebase 연결 버튼으로 공가앱 데이터 실시간 연동
• 공가앱에서 세대 상태 변경 → 하자앱에 자동 반영
• 🏠 공가미해결: 공가앱에서 공급불가 상태인 세대 중 코티스 미처리 하자 있는 세대
• 세대 상세에서 공가 현황(공가일수·전출일·불가사유)도 함께 표시
