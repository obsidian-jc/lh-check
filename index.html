📱 로컬 백업 (기기 저장)
인터넷 없이도 사용 가능 · 기존 방식
💻 윈도우 PC에서 사용하기
스마트폰에서 작업한 데이터를 PC에서 이어서 사용하는 방법입니다.
1 📱 스마트폰에서 백업 저장
위 💾 백업 파일 저장 버튼을 눌러 JSON 파일을 저장합니다.
파일명: LH공가점검_백업_날짜.json
2 📤 PC로 파일 전송 (택 1)
추천 카카오톡 나에게 보내기 — 파일 첨부 → 내 채팅 → PC 카카오톡에서 다운로드
간편 이메일 첨부 — 본인 이메일로 파일 첨부 발송
USB USB 케이블 — 스마트폰 연결 후 파일 복사
3 💻 PC에서 앱 파일 열기
PC에서 index.html 파일을 더블클릭 하거나,
Chrome/Edge 주소창에 파일 경로를 입력해 실행합니다.
⚠️ Internet Explorer는 지원하지 않습니다.
4 📂 백업 파일 불러오기
PC에서 앱이 열리면 내보내기 탭 → 📂 백업 파일 불러오기를 눌러 JSON 파일을 선택합니다.
✅ 스마트폰의 모든 점검 기록이 PC에 동기화됩니다!
🔄 PC 작업 후 스마트폰으로 돌아올 때
PC에서 작업 완료 → 백업 저장 → 카카오톡/메일로 전송
스마트폰에서 백업 불러오기 → 최신 데이터로 복원 완료
